About This Demo

            
This application demonstrates:

            
    
                
  • Static Web App: Hosting this frontend application
    • 
                
  • Container App: Public API running in a container from ACR
    • 
                
  • Function App: Private backend service (VNet integrated)
    • 
                
  • Azure SQL: Database with managed identity authentication
    • 
                
  • Storage Account: Blob storage with private endpoint
    • 
                
  • Key Vault: Secrets management with RBAC
    • 
                
  • ACR: Private container registry
    • 
                
  • Managed Identity: Secure, keyless authentication throughout
